# Personal Book & Movie Collection Manager

A Lightweight Flask Web Application built as a Software Engineering final project. It helps users register, log in, and securely manage their personal book and movie collections with raw SQL queries and data persistence.

## 🚀 Features

- **User Authentication:** Secure Sign-Up, Log-In, and Log-Out system using session management and dynamic password hashing (`[US1]`, `[US2]`).
- **Full CRUD Operations:** Users can Create, Read, Update, and Delete items in their collection (`[US1]`, `[US2]`, `[US3]`).
- **Business Logic & Analytics:** Interactive stats dashboard showcasing total books, total movies, and global average rating.
- **Advanced Frontend:** Light-themed UI with client-side dynamic search and type filtering (All / Books / Movies) without refreshing the page.
- **Security First:** Robust architecture utilizing central login-required decorators to protect private routes.

## 🛠️ Tech Stack

- **Backend:** Python / Flask
- **Database:** SQLite3 (Raw SQL Queries)
- **Frontend:** HTML5 / CSS3 / Bootstrap 5 / Vanilla JavaScript
